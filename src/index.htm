Pacotes Snap

Pacotes snap são programas que são baixados e instalados via linha de comando e que são instalados em qualquer distro que possua soporte ao snap.

Para saber se sua distro ja vem com os pacotes instalados diretamente digite o comando:

$ snap --version

Caso não apareça uma mensagem como essa:

$ snap --version
    snap    2.38.1
    snapd   2.38.1
    series  16
    ubuntu  18.04
    kernel  4.18.0-18-generic
    $

Caso não apareça instale com o comando, conforme sua distro:

  • Ubuntu e derivados:

    $ sudo apt update 
    $ sudo apt install snapd -y

  • Arch e derivados:

    $ git clone https://aur.archlinux.org/snapd.git
    $ cd snapd
    $ makepkg -si
    $ sudo systemctl enable --now snapd.socket
    $ sudo ln -s /var/lib/snapd/snap /snap

  • Manjaro:

    $ sudo pacman -S snapd
    $ sudo systemctl enable --now snapd.socket
    $ sudo ln -s /var/lib/snapd/snap /snap

  • CentOS e derivados:

    $ sudo yum install epel-release
    $ sudo yum install snapd
    $ sudo systemctl enable --now snapd.socket
    $ sudo ln -s /var/lib/snapd/snap /snap

  • Fedora e derivados: 

    $ sudo dnf install snapd
    $ sudo ln -s /var/lib/snapd/snap /snap

  • OpenSUSE

    $ sudo zypper addrepo --refresh 
    https://download.opensuse.org/repositories/system:/snappy/openSUSE_Leap_15.0 snappy
    ...
    Repository 'snappy' successfully added
    $ sudo zypper --gpg-auto-import-keys refresh
    $ sudo zypper dup --from snappy
    $ sudo zypper install snapd
    $ sudo systemctl enable snapd   
    $ sudo systemctl start snapd
    $ sudo systemctl enable snapd.apparmor
    $ sudo systemctl start snapd.apparmor
    $ sudo zypper removerepo snappy
    $ sudo zypper addrepo --refresh 
    https://download.opensuse.org/repositories/system:/snappy/openSUSE_Leap_15.0 snappy

  • Solus Os:

    $ sudo eopkg up
    $ sudo eopkg install snapd

Pacotes Flatpak

Pacotes Flatpak são programas que são baixados e instalados via linha de comando e que são instalados em qualquer distro que possua soporte ao snap.

Para saber se sua distro ja vem com os pacotes instalados diretamente digite o comando:

$ flatpak --version

Caso não apareça uma mensagem como essa:

$ flatpak --version
    Flatpak 1.0.8
    $

Caso não apareça instale com o comando, conforme sua distro:

  • Ubuntu e derivados:

    $ sudo add-apt-repository ppa:alexlarsson/flatpak
    $ sudo apt update
    $ sudo apt install flatpak
    $ sudo apt install gnome-software-plugin-flatpak
    $ flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
    $ reboot

  • Redhat e baseados:

    $ sudo yum install flatpak
    $ flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
    $ reboot

  • Fedora e derivados:

    $ flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
    $ reboot

  • OpenSUSE e derivados:

    $ sudo zypper install flatpak
    $ flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
    $ reboot

  • Arch e baseado:

    $ sudo pacman -S flatpak
    $ reboot

Shell Script

Shell Script é uma linguagem de programação baseada nos comandos de linux, e assim, com um simples arquivo com comandos e as estruturas básicas de uma linguagem de programação (if, while, for...).

Teremos também scripts feitos por nos para automatizar instalações.

Veja os escripts criados por nos:

Instalador para desenvolvedores

#!/bin/bash

zenity --info 	\
    --title="Iniciando" \
    --text="Olá, iniciando o instalador!" \
    --width=250 \
    --height=30

lista=$(zenity --list \
    --title="Programas" \
    --text="Selecione os programas:" \
    --column="Selecione" --column="Programa" \
    false "PyCharm Community" \
    false "Atom" \
    false "Notepad++" \
    false "Android Studio" \
    false "Php Storm" \
    false "Eclipse" \
    false "PyCharm Pro" \
    false "IDEA Community" \
    false "Notpadqq" \
    false "Brackets" \
    false "IDEA Community" \
    false "Webstorm" \
    --checklist)

if [ -z "$lista" ]; then
    zenity 	--error \
        --title="Alerta!" \
        --text="Nenhum item selecionado!" \
        --width=250 \
            --height=30
    exit 1
fi

comando=""
IFS="|"; for programa in lista ; do
    case $programa in
        "PyCharm Community") comando="${comando} \n sudo snap install pycharm-community"  ;;
        "Atom") comando="${comando} \n sudo snap install atom" ;;
        "Notepad++") comando="${comando} \n sudo snap install notepad-plus-plus" ;;
        "Android Studio") comando="${comando} \n sudo snap install android-studio" ;;
        "Php Storm") comando="${comando} \n sudo snap install phpstorm" ;;
        "Eclipse") comando="${comando} \n sudo snap install eclipse" ;;
        "PyCharm Pro") comando="${comando} \n sudo snap install pycharm-professional" ;;
        "IDEA Community") comando="${comando} \n sudo snap install intellij-idea-community" ;;
        "Notpadqq") comando="${comando} \n sudo snap install notepadqq" ;;
        "Brackets") comando="${comando} \n sudo snap install brackets" ;;
        "IDEA Community") comando="${comando} \n sudo snap install intellij-idea-ultimate" ;;
        "Webstorm") comando="${comando} \n sudo snap install webstorm" ;;
    esac
done

zenity --info 	\
    --title="Comando" \
    --text="$comando" \
    --width=250 \
    --height=30
